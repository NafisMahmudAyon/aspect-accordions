### Description

**Aspect Accordions** is a powerful and user-friendly plugin for creating dynamic accordion layouts, FAQ sections, tabs, and tabbed content. Ideal for use cases such as:  
- Frequently Asked Questions (FAQs)  
- Knowledge Bases  
- Question and Answer sections  
- WooCommerce FAQ tabs  

With an intuitive builder, customizing the look and feel has never been easier. Change colors, fonts, and icon styles with just a few clicks to perfectly match your site's design.

#### Key Features:  
1. **Accordion and Tab Builder:**  
   Create accordions and tabs visually with a modern builder interface.  
2. **Dynamic Customization Options:**  
   Easily adjust the appearance, including color schemes, font sizes, and icons.  
3. **Shortcode Integration:**  
   Add your accordion or tab layouts anywhere on your site using shortcodes.  
4. **Seamless Page Builder Support:**  
   Integrate with popular page builders for enhanced flexibility.  

---

### Builder Interface

This plugin introduces a standalone builder for creating and managing accordions and tabs. Unlike traditional plugins that rely heavily on WordPress's default dashboard UI, the **Aspect Accordions** builder provides a sleek and interactive interface at:  
**`http://localhost/wp/wp-admin/admin.php?page=aspect-accordions`**  

Here, you can:  
- Design and preview your accordion or tab layouts in real time.  
- Manage all customization options from a single, intuitive interface.  
- Avoid navigating between multiple pages or sections in the WordPress dashboard.
